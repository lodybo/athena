# Button
A button. Especially useful for user-initiated actions.

<Canvas>
  <Story
    name="A button"
  >
    { ButtonTemplate.bind({}) }
  </Story>
</Canvas>

A button doesn't need to have background and border styles.
A simple button can be summoned.

<Canvas>
  <Story
    name="Simple button"
    argTypes={{
      argTypes: argTypes.simple
    }}
    args={{
      simple: true,
    }}
  >
    { ButtonTemplate.bind({}) }
  </Story>
</Canvas>

Instead of a transparent background, a button can also have a solid fill.

<Canvas>
  <Story
    name="Primary button"
    args={{
      solid: true,
    }}
  >
    { ButtonTemplate.bind({}) }
  </Story>
</Canvas>

It's also possible to determine which theme will be used for the button.

<Canvas>
  <Story
    name="Theming the button"
    argTypes={{
      argTypes: argTypes.theme
    }}
    args={{
      theme: 'tertiary',
    }}
  >
    { ButtonTemplate.bind({}) }
  </Story>
</Canvas>

If needed, you can pass other colors to it in the form of a [classnames](https://www.npmjs.com/package/classnames) argument.

<Canvas>
  <Story
    name="Specifying a class"
    args={{
      classes: 'example-class'
    }}
  >
    { ButtonTemplate.bind({}) }
  </Story>
</Canvas>

# Button
A button. Especially useful for user-initiated actions.

<Canvas>
  <Story
    name="A button"
  >
    { ButtonTemplate.bind({}) }
  </Story>
</Canvas>

A button doesn't need to have background and border styles.
A simple button can be summoned.

<Canvas>
  <Story
    name="Simple button"
    argTypes={{
      simple: argTypes.simple,
    }}
    args={{
      simple: true,
    }}
  >
    { ButtonTemplate.bind({}) }
  </Story>
</Canvas>

Instead of a transparent background, a button can also have a solid fill.

<Canvas>
  <Story
    name="Primary button"
    argTypes={{
      solid: argTypes.solid,
    }}
    args={{
      solid: true,
    }}
  >
    { ButtonTemplate.bind({}) }
  </Story>
</Canvas>

It's also possible to determine which theme will be used for the button.

<Canvas>
  <Story
    name="Theming the button"
    argTypes={{
      theme: argTypes.theme
    }}
    args={{
      theme: 'tertiary',
    }}
  >
    { ButtonTemplate.bind({}) }
  </Story>
</Canvas>

If needed, you can pass other colors to it in the form of a [classnames](https://www.npmjs.com/package/classnames) argument.

<Canvas>
  <Story
    name="Specifying a class"
    args={{
      classes: 'example-class'
    }}
  >
    { ButtonTemplate.bind({}) }
  </Story>
</Canvas>

## Icons
A button also supports displaying icons, which can be especially useful if you'd like to have a bar of buttons.
Plus, sometimes an icon can say a thousand words.

Athena has Font Awesome integrated, so you can add icons to the library and reference them in components.

<Canvas>
  <Story
    name="Displaying icons"
    argTypes={{
      icon: argTypes.icon,
      simple: argTypes.simple,
      solid: argTypes.solid,
    }}
    args={{
      icon: 'check',
      simple: false,
      solid: false,
    }}
  >
    { ButtonTemplate.bind({}) }
  </Story>
</Canvas>

Of course its possible to determine at which side of the button the icon will be displayed:

<Canvas>
  <Story
    name="Positioning icons"
    argTypes={{
      icon: argTypes.icon,
      iconPlacement: argTypes.iconPlacement,
      simple: argTypes.simple,
      solid: argTypes.solid,
    }}
    args={{
      icon: 'check',
      iconPlacement: 'right',
      simple: false,
      solid: false,
    }}
  >
    { ButtonTemplate.bind({}) }
  </Story>
</Canvas>

# Alert
An `<Alert />` can showcase a notification to a user. This can either be a notification, or a reaction to a validation status.
The colors follow either the primary color scheme, or the colors for the success, warning and error schemes.

## Hiding the `<Alert />`
One of the states you can pass to the `<Alert />` (which is also enabled by default) is the `HIDDEN` state.
This allows the `<Alert />` to be managed from a parent component.

Since the `<Alert />` has a close button that the user can click. You can use the `onHide` prop to pass a callback function that can
change the state.
